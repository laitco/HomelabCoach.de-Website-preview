Remote-Support

Direkte Hilfe – sicher & verschlüsselt

Der Remote-Support läuft über meinen eigenen RustDesk-Server. Die Verbindung ist direkt und verschlüsselt – ohne Drittanbieter, ohne dass deine Daten über externe Server laufen.

Sicherer Remote-Support

Privater Server

Die Verbindung läuft ausschließlich über meinen eigenen Server – keine fremden Dienste.

Ende-zu-Ende-verschlüsselt

Alle Verbindungen sind verschlüsselt. Niemand kann den Datenverkehr mitlesen.

Keine Installation nötig

LAITCO-HELP ist eine portable App – einfach starten, keine Installation erforderlich.

So funktioniert es

App herunterladen

Lade LAITCO-HELP für dein Betriebssystem herunter (siehe unten).

LAITCO-HELP starten

Starte die App – keine Installation, einfach doppelklicken.

ID & Passwort teilen

Teile mir die angezeigte ID und das einmalige Passwort – ich verbinde mich dann.

LAITCO-HELP herunterladen

Wähle die Version für dein Betriebssystem.

Windows 10/11 (x64)

laitco-help.exe herunterladen

macOS – Apple Silicon (M1–M4)

laitco-help-arm64.dmg herunterladen

macOS – Intel

laitco-help-x64.dmg herunterladen

Linux – ARM (aarch64)

laitco-help-aarch64.deb herunterladen

Linux – x86/x64

laitco-help-x86_64.deb herunterladen

Android (APK)

laitco-help.apk herunterladen

Hinweis für macOS: Nach dem ersten Start musst du in den Systemeinstellungen unter Datenschutz & Sicherheit die Berechtigungen für Bildschirmfreigabe und Eingabesteuerung erteilen, damit der Remote-Support funktioniert.
